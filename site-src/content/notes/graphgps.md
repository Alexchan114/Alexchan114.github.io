GraphGPS 提供了一个很清楚的拆解视角：图模型不必在“纯 GNN”和“纯 Transformer”之间二选一，而可以分别处理局部结构和全局信息。

## 三个需要分开的变量

- **结构或位置编码**：告诉模型节点处于怎样的图结构位置。
- **局部消息传递**：利用邻接关系形成强归纳偏置。
- **全局注意力**：让远距离节点直接交换信息。

如果三部分一次性打开，即使指标提高，也很难判断贡献来自哪里。因此实验配置应当把它们作为独立轴进行消融。

## 最小实验矩阵

| 组别 | 局部层 | 全局层 | 结构编码 |
| --- | --- | --- | --- |
| A | GCN | 无 | 无 |
| B | GCN | Attention | 无 |
| C | GCN | 无 | 开启 |
| D | GCN | Attention | 开启 |

在 ZINC 等小分子图任务上，除了最终误差，还要观察参数量、收敛速度和不同随机种子的波动。只有评价成本保持可比，组合架构的收益才有解释价值。
